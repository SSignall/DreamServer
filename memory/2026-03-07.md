### ComfyUI AMD Compose Security ⚠️
- AMD compose has `seccomp:unconfined` and `SYS_PTRACE` cap
- These are required for AMD GPU access (`/dev/dri`, `/dev/kfd`)
- Documented as known security tradeoff — cannot remove without breaking AMD support
