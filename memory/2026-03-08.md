## Security Pattern Reference (All n8n Workflows)

**timingSafeEqual Implementation:**
```javascript
function timingSafeEqual(a, b) {
  if (typeof a !== 'string' || typeof b !== 'string') return false;
  const MAX_LEN = 256;
  if (a.length > MAX_LEN || b.length > MAX_LEN) return false;
  
  let result = 0;
  for (let i = 0; i < MAX_LEN; i++) {
    const charA = i < a.length ? a.charCodeAt(i) : 0;
    const charB = i < b.length ? b.charCodeAt(i) : 0;
    result |= charA ^ charB;
  }
  result |= (a.length === b.length) ? 0 : 1;
  return result === 0;
}
```

**Key Requirements:**
- 256-char max for timing-safe comparison
- Early return on type mismatch
- Include length check in constant-time result
- No `Math.max(a.length, b.length)` (allows length oracle attacks)

**Discord Node Parameter:**
- Must use `webhookUri` (not `channel`)
- Format: `={{ $env.DISCORD_WEBHOOK_URL || '' }}` or dynamic expression
