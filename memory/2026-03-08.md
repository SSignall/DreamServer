## Commit Review: macOS JSON Escape Fix

### Commit `8293cf4e`
**Issue:** Incomplete JSON escape sequences in macOS env-generator.sh fallback

**Fix:** Added complete RFC 8259 mandatory escapes:
- `\b` (backspace)
- `\f` (form feed)
- `\n` (newline)
- `\r` (carriage return)
- `\t` (tab)
- `\\` (backslash)
- `\"` (quote)

**Defense in depth:** Escape all control chars even for hex tokens

**Pattern:** Use `$'\b'` and `$'\f'` for literal control chars in BSD sed

### Security Note
Proper JSON escaping prevents injection attacks via token strings. RFC 8259 compliance is mandatory for token storage in JSON config files.
