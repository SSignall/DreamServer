## Fix: Whisper Duplicate Patch Bug (Issue #57)

**Problem:** Container restart loop with `SyntaxError: keyword argument repeated: vad_parameters`

**Root Cause:** The regex pattern `vad_filter\s*=\s*effective_vad_filter` matched even after patching because the patched line still contained that substring. Each restart added another `vad_parameters` argument.

**Fix:** Pattern now requires end-of-line after `effective_vad_filter`:
```
vad_filter\s*=\s*effective_vad_filter[[:space:]]*,?[[:space:]]*$
```

**Commit:** `958c5cd7`
